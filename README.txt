To run the application, find the Visual Studio Solution file titled
"MaritimeDatabase.sln" in the Maritime Database file directory, once found
open this file in Visual Studio.

The solution should open containing the MaritimeApplication and the
MaritimeDatabase.

To make sure that the database exists and can connected to, the project will
need to be ran and deployed inside the solution.
Once this is done, the MaritimeApplication will be able to be run and the
Web Application should open in your browser.
From there, upload the SampleData CSV file to the application.
